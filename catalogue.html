Your Courses

Introduction to Programming

This course provides an introduction to programming using Python. Topics covered include data types, control structures, functions, arrays, and file processing.

Data Structures and Algorithms Lab

This course covers basic data structures such as arrays, lists, stacks, queues, and trees, as well as fundamental algorithms such as sorting and searching. Students will implement these data structures and algorithms using Python.

Database Systems

This course provides an introduction to database systems and their design. Topics covered include relational data model, SQL, normalization, indexing, and transaction management.

Web Development Lab

This course covers front-end web development using HTML, CSS, and JavaScript, as well as back-end web development using PHP and MySQL.

Computer Networks

This course introduces the fundamentals of computer networks and their applications in modern communication systems.

Soft Computing Lab

This course is designed to provide hands-on experience with various techniques and tools of soft computing, such as neural networks, fuzzy logic, and genetic algorithms.

Operating Systems

This course provides an overview of the fundamental concepts and principles of operating systems, including process management, memory management, file systems, and security.

Full Stack Development Lab

This course is designed to provide hands-on experience with building web applications using a full stack of technologies, including HTML, CSS, JavaScript, Node.js, and databases.
